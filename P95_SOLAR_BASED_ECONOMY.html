SOLAR_BASED_ECONOMY

PART_0: Physical Quantities of the System

photons: a natural number quantity which represents some empirically observed quantity of photons (i.e. irreducible particles of light).

electrons: a natural number quantity which represents some empirically observed quantity of electrons (i.e. irreducible particles of electricity).

E: a natural number representing the average daily number of joules of energy generated by all functional photovoltaic cells in the Solar Based Economy system.

let E = electricity_in_joules(electrons); 
// electrons are from photovoltaic cells
// E represents total photovoltaic output generated during a particular year.

P: a natural number representing the total number of end users (i.e. persons) of the Solar Based Economy system.

P: a natural number representing the total number of personal bank accounts which serve as decentralized end points of the Solar Based Economy system.

(Each end user of the Solar Based Economy system has exactly one unique personal bank account which is not shared with any other user of the system).

PART_1: Abstract Quantities of the System

A: an arbitrary natural number which is used as a scaling coefficient.

N: an arbitrary natural number which represents the total number of Solar Energy Tokens within the system.

(N is a function which takes E as an input and which rounds E down to the nearest integer (i.e. a whole number greater than or equal to zero) and multiplied by A.

let N = round_down_to_nearest_integer(E * A); 
// N is the total number of tokens created during a particular year

T: an arbitrary natural number which represents the minimum number of Solar Energy Tokens to newly generate and consequently distribute to each one of the P personal bank accounts which serve as decentralized end points of the Solar Based Economy system at midnight on the first day of each new year.

let T = (N / P); 
// T is the annual basic income for each person

PART_2: Logical Features of the System

With each successive year, (T * P) new Solar Energy Tokens are added to the Solar Based Economy system such that the total number of tokens increments by roughly N per year.

N += N; 
// closed system simulation
N = (N + N); 
// closed system simulation
N += round_down_to_nearest_integer(E * A); 
// empirical input, E, introduces randomness
N = N + round_down_to_nearest_integer(E * A); 
// empirical input, E, introduce randomness

The number of electrons (i.e. irreducible particles of electricity) which are ejected from a photovoltaic cell are directly proportional to the number of photons (i.e. irreducible particles of light) which strike the ionic surface of the photovoltaic cell.

let electrons = photovoltaic_conversion(photons); 
// photons are from sunlight
let E = electricity_in_joules(electrons); 
// electrons are from photovoltaic cells
